ctrl+alt+t - open terminal

ls - просмотр содержимого текущей директории
ls путь_до_директории - prosmotr soderjimogo ekazannoi direktorii
ls -a - просмотр скрытых файлов директории

mkdir - название_директории
cd путь_до_директории - смена директории

"." - ссылка на текущую директорию
".." - ссылка на директорию выше/родительскую директорию
"~" - ссылка на домашнюю директорию

pwd - вывод абсолютного пути в консоль

Абсолютный путь - путь, начинающийся с домашней директории (home/fnizamov/some_dir)
Относительный путь - путь, относительно других директорий (./some_dir)

tree

rmdir путь_до_директории - удаление пустой директории
rm -rf имя_директории - принудительное удаление директории с содержимым

touch - создание файла в текущей директории
touch some_dir - создание файла в указанной директории/по указанному пути
rm имя_файла - удаление файла
rm -rf имя_файла - принудительное удаление файла

cat имя_файла - вывод содержимого файла в терминал
cat имя_файла >> имя_другого_файла - дозапись содержимого одного файла в другой файл

echo "какой-то текст" - вывод указанного текста в терминал
echo "какой-то текст" >> название_файла - дозапись указанного текста в файл

mv имя_файла новое_имя_файла - переименование файла
mv имя_файла путь_до_директории - перенос файла в указанную директорию

nano путь_до_файла - открытие файла в текстовом редакторе nano. Если указанного файла не существует, то создает его.

первая_команда && последующая_команда - объединение первая_команда

cp - копирование файлов из одной директории в другую
